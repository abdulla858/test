# DSA Master - Data Structures & Algorithms Quiz

A premium, interactive web application to test and improve your knowledge of Data Structures and Algorithms.

## Features

- **Comprehensive Quiz Categories**: Arrays, Stacks, Queues, Linked Lists, Trees, Searching Algorithms, and Time Complexity.
- **Bilingual Support**: Full support for English and Arabic languages.
- **RTL Support**: Seamless Right-to-Left (RTL) layout for Arabic users.
- **Modern Aesthetics**: Sleek dark/light modes with glassmorphism and smooth animations.
- **Detailed Explanations**: Every question includes a clear explanation to help you learn.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## How to Run

Simply open `index.html` in your web browser.

## Project Structure

- `index.html`: The main entry point.
- `app.js`: Application logic and state management.
- `data.js`: Quiz database with translated content.
- `style.css`: Design tokens and responsive layouts.
